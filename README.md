# TenHearts ❤
________________
## A Stardew Valley villager compatibility quiz.
Taking inspiration from buzzfeed questions, take a quiz to find out the most compatible bachelor or bachelorette for you!

## How It Works
Answer introduction questions and pick your favorite from a randomized choices. Scores are based on villager's specific loved, liked, disliked, and hated gifts. 


## Tech Stack
- Python
- Pandas
- Jupyter Notebook

## How To Run
1. Clone the repo
2. pip install -r requirements.txt
3. python run utils.py

## Data Source
Data sourced from Official Stardew Valley Wiki. Data sets are collected and cleaned manually.
